Frontend

Backend

O backend está armazenado no Heroku e segue as especificações REST, quanto ao nome dos endpoinst. O código fonte está no GitHub.

A URL base é: https://bocelli-app.herokuapp.com/api e os serviços oferecidos são:

GET: /post/

Retorna todos os posts.

Requer: Nenhum parâmetro;

GET: /post/:id/

Retorna o post com o id passado como parâmetro.

Requer:

  • id - String contendo o id do post selecionado (obrigatório);
POST: /post/

Cria um novo post.

Requer:

  • title - String contendo o título do post (obrigatório);
  • description - String contendo o conteúdo do post (obrigatório);
  • creator - String contendo o autor do post (obrigatório);
PUT: /post/:id/

Substitui as informações do post com o id passado como parâmetro.

Requer:

  • id - String contendo o id do post selecionado (obrigatório);
  • title - String contendo o título do post (obrigatório);
  • description - String contendo o conteúdo do post (obrigatório);
  • creator - String contendo o autor do post (obrigatório);
DELETE: /post/:id/

Deleta o post com o id passado como parâmetro.

Requer:

  • id - String contendo o id do post a ser deletado (obrigatório);
